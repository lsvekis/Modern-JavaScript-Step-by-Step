# JavaScript by Building — 250 Exercises (GitHub Pages v2)

This repository is **GitHub Pages-ready** using the `/docs` folder.

## Publish
1. Push to GitHub
2. Settings → Pages
3. Source: Deploy from a branch
4. Branch: main, Folder: /docs

Then open the Pages URL and you’ll have:
- Searchable index
- Next/Prev navigation per exercise
- Visible UI progression (not only console logs)
